CelluCheck adalah aplikasi web screening awal untuk mendeteksi kemungkinan cellulitis berdasarkan analisis gambar kulit dan kuesioner gejala medis. Dirancang khusus untuk edukasi masyarakat dan demo lomba teknologi kesehatan.

  • Flask: Framework web Python untuk backend
  • Bootstrap: UI/UX responsif dan modern
